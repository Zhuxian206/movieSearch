## 使用說明
機器人有提供「電影搜尋」、「新片推薦」、「查看周末票房名次」等功能。
如有想要搜尋的電影，可直接手動輸入片名關鍵字。

輸入「!new」，會隨機向您推薦五部本周新片。
輸入「!p」，則會顯示上周末的票房名單。
想查看使用方式請輸入「!說明」。

也可點選畫面下方選單使用以上功能。
